<img width="1920" height="1020" alt="image" src="https://github.com/user-attachments/assets/9d5ce609-14c7-4f65-b1f2-49f7782f3728" /># Markdown 编辑器

基于 Flask + Vue3 的在线 Markdown 编辑器系统。

## 功能特性

- ✅ 实时 Markdown 编辑
- ✅ 同步预览渲染后的 HTML
- ✅ 导出为 .md文件
- ✅ 响应式设计，支持移动端

## 技术栈

**后端：**
- Flask 3.0
- Flask-CORS
- Markdown
- Python 3.8+

**前端：**
- Vue 3
- Vite 5.0
- Axios
- Marked

## 快速开始

### 环境要求

- Python 3.8+
- Node.js 16+

### 启动项目之前先启动前端部分
```
npm run dev
```
### 打开http://localhost:3000/
